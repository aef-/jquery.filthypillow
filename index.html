jquery.filthypillow

a fancy and small calendar and date-time picker

Download .zip Download .tar.gz View on GitHub

Requirements

Tested with:
moment 2.2.1
jquery 1.9.1

Examples

{% highlight javascript %} var $fp = $( ".filthypillow-1" ); $fp.filthypillow( { minDateTime: function( ) { return moment( ); //now } } ); $fp.on( "focus", function( ) { $fp.filthypillow( "show" ); } ); $fp.on( "fp:save", function( e, dateObj ) { $fp.val( dateObj.format( "MMM DD YYYY hh:mm A" ) ); $fp.filthypillow( "hide" ); } ); {% endhighlight %}

Hotkeys

enter (save)

left-right arrows/tab (switch steps)

up-down arrows (change digits)

number keys (change digits)

Configuration

startStep

Step that should be active on initialization.

month, day, hour, minute, meridiem

Default

day

initialDateTime

Set the initial date time to be shown.

Default

null

Example

{% highlight javascript %} initialDateTime: function( m ) { return m.add( "day", 7 ); } {% endhighlight %}

minDateTime

Set the earliest time that can be selected.

Default

null

Example

{% highlight javascript %} minDateTime: function( ) { return moment( ); //now } {% endhighlight %}

maxDateTime

Set the latest time that can be selected.

Default

null

Example

{% highlight javascript %} maxDateTime: function( ) { return moment( ).add( "days", 7 ); //7 days from now } {% endhighlight %}

Events

fp:save

Passes selected date object.

{% highlight javascript %} $( ".datetime" ).filthypillow( ); $( ".datetime" ).on( "fp:save", function( e, dateObj ) { console.info( dateObj ); } ); {% endhighlight %}
